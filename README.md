<h2> &#9881; Instalação do Projeto</h2>
Se quiser instalar na sua máquina para se inspirar, rode os comandos abaixo:

   ```bash
   git clone https://github.com/vpaesi/vitoria-de-camargo.git
   ```

   ```bash
   cd vitoria-de-camargo
   npm install
   npm run dev
   ```

Acesse:
`http://localhost:5173`
